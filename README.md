### ЗАДАЧА
реализовать функции для поиска подстрок по следующим алгоритмам:

- Бойера-Мура; 
- Рабина-Карпа; 
- Кнута-Морриса-Пратта;


#### собрать проект
```bash
$ mkdir build
$ cd build/
$ cmake ..
$ make 
```

#### запустить тестирование
```bash
./test/lab6_TEST
```

#### запустить benchmark
```bash
./benchmark/lab6_substr_serch_BENCHMARK
```
